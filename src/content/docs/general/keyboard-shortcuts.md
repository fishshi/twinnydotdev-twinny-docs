---
title: Keyboard shortcuts
description: Keyboard shortcuts for twinny.

---

| Shortcut                    | Description                                      |
| ----------------------------| -------------------------------------------------|
| `ALT+\`                     | Trigger inline code completion                   |
| `CTRL+SHIFT+/`              | Stop the inline code generation                  | 
| `Tab`                       | Accept the inline code generated                 |
