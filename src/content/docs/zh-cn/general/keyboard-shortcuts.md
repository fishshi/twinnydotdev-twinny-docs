---
title: 键盘快捷键  
description: Twinny 的键盘快捷键  

---

| 快捷键                        | 描述                                              |
| ----------------------------- | ------------------------------------------------- |
| `ALT+\`                        | 触发内联代码补全                                  |
| `CTRL+SHIFT+/`                 | 停止内联代码生成                                  |
| `Tab`                          | 接受生成的内联代码                                |
